!!! RUN THIS SETUP AS ADMINISTRATOR !!!


This is the installation file for the IDEA AddOn. 
Please note that this AddOn needs to be activated in the Manage AddOn screen in Enterprise Architect
This AddOn is published under the EUPL Open Source License

There are four extra files for more functionality
* IDEAResources.xml Import this via EA with Import Reference data, for scripts, report templates and DD setups. This file is relevant for the users who do not want to use the AddOn in the DLL 
* BizzTemplate.xlsx is a template for the BizzDesign format to import via the assistant screen in IDEA
* htmltemplate.zip, a sample setup for the HTML generator in the addon, to be configured via the screen in the AddOn
* IDEAEaxmple.qea an example repository developed for the EA Global Summit but also relevant for other IDEA users
* Text2ArchiMate.sql a SQL Server stored procedure set for textmining based on ArchiMate descriptions

This new version includes the LDM Simulator for Open Oriented Data Modeling, please find more information on this module on the webpage below.

Please be aware that there is both a 32 and 64 bits version available for EA. Please see the number in the zip file. The 32 bits version can ben updated on request. please send me an email
