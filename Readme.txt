Installatie IDEA addon

Plaats de bestanden uit de zip in een folder
Registreer de dll in regasm met
RegAsm.exe "volledige filenaam" /codebase

Importeer de reg sleutel in de registry 
(letop dit kan anders zijn omdat het voorbeeld uitgaat van een local user setting)
Voor gebruikers van de oudere versie de TEA registry key kan weg
De map met extra files dienen per repository geinstalleerd te worden via import resources in EA

Voor meer informatie zie http://eaxpertise.nl/cmsform.aspx?webpage=TEA
